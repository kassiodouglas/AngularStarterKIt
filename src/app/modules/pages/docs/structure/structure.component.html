Estrutura

Segue mapa da estrutura e suas explicações.

assets/ => contem todos os arquvios estaticos como fontes, estilos, imagens

environments/ => contem os ambientes e suas variaveis

app/ => pasta base contendo os arquivos da aplicação

app/routes => rotas da aplicação

app/services => serviços ,em geral, para consumo de dados do backend

app/models => local para armazenar as interfacer de dados

app/core => nucleo da aplicação com classes abstratas [?]

app/core/app => componente base de toda a aplicação

app/core/commands => armazenamdneot dos comandos internos e novos

app/core/directives => diretivas internas da aplicação [?]

app/core/guards => contem os middlewares para interceptar rotas

app/modules => contem todos os componentes da aplicação

app/modules/components => componentes 'compartilháveis' na aplicação ('pedaços' de códigos reutilizáveis)

app/modules/layouts => componentes de 'interface' contendo elemetos padrão de locais da aplicação, como navegação, footer...

app/modules/pages => componentes 'páginas' que englobam sub-componentes
