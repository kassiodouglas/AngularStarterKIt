Angular 16.0 Starter Kit

Instalado de fábrica:

  • ng-bootstrap 15.1.0
  • Bootstrap Icons 1.10.5
  • Notiflix 3.2.6

    • Comandos de fábrica:

  • ng serve
    Servidor de desenvolvimento
  • npm run cs -- --name nameService
    Criação de serviços
  • npm run ci -- --name nameInterface
    Criação de interfaces
  • npm run cc -- --local name/dir --name nameComponent
    Criação de componentes
